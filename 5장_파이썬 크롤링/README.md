# Crawling and word-cloud

## Crawling
### Pre-requisties
|  <center>Requirement</center> |  <center>Description</center> |  
|:--------|:--------:|
|**Python** | <center>v3.X or higher</center> |
|**mecab** | <center>v0.996 or higher</center> |

```
pip install python-mecab-ko
```
